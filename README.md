# 🐍 Snake Game

A classic Snake Game built using HTML, CSS, and JavaScript. Control the snake, eat food, grow longer, and try to achieve the highest score without colliding with the walls or yourself.

## 🚀 Features

- Smooth snake movement
- Score tracking system
- Food generation
- Game over detection
- Keyboard controls
- Responsive game layout
- Simple and clean UI

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

## 🎮 How to Play

1. Use the arrow keys to move the snake.
2. Eat the food to grow longer.
3. Avoid hitting:
   - Walls
   - Snake's own body
4. Try to score as high as possible.

## ⌨️ Controls

| Key | Action |
|------|--------|
| ↑ | Move Up |
| ↓ | Move Down |
| ← | Move Left |
| → | Move Right |

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/pdpreeti/Snake-Game.git
```

Open the project folder:

```bash
cd Snake-Game
```

Then open `index.html` in your browser.
